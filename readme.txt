Verdienen Sie Geld mit Context Link Anzeigen in Ihrem Blog

== Description ==

conliad ist eine vollautomatischer Text-Link Marktplatz. Verwenden Sie das conliad Plugin um kontextuelle Links aus Ihrem Blog zu verkaufen. Sobald Sie das Plugin installiert haben übernehmen wir den Rest!

Ihr Blog wird, falls gewünscht anonymisiert, auf dem Marktplatz potientellen Linkkäufern angeboten und diese können dann den Link bei Ihnen buchen. 

== Installation ==

Folgen Sie einfach der folgenden Anleitung:

1. Entpacken Sie alle Dateien von `conliad.zip` in Ihr Plugin-Verzeichnis (`/wp-content/plugins/`)
2. Aktivieren Sie das Plugin im Wordpress-Administrationsbereich unter 'Plugins'
3. Gehen Sie auf conliad.de, regestrieren Sie sich und holen Sie sich den Token füe 

Webseite
4. Gehen Sie zu den 'conliad Einstellung' im Wordpress und kopieren Sie den Token rein
